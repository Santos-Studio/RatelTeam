# RATEL TEAM

Site vitrine pour le club de Jiu-Jitsu Brésilien de Rennes **RATEL TEAM**.

## Description

Site web présentant le club de Jiu-Jitsu Brésilien RATEL TEAM basé à Rennes. Le site présente les cours, les membres, les dojos, le planning et toutes les informations nécessaires pour rejoindre le club.

## Technologies utilisées

- **HTML** - Structure du site
- **CSS** - Styles et design responsive
- **JavaScript** - Interactivité et animations
